# Travel Agency

Travel Agency client/server applications

## TServer
### Usage
```
talent_server ip_address start_port no_ports data_file output_file
```

### Commands
#### EXIT 
    Exits the server.

#### LIST 
    List all users connected.

#### LIST CHAT 
    List all users in chat.

#### WRITE `filename` 
    Write `filename` with current data.
